Installation & Configuration

Comment inviter YAKO sur mon serveur ?

Pour inviter YAKO sur votre serveur Discord :

  1. Cliquez sur le bouton "Ajouter à Discord" sur notre site
  2. Connectez-vous à votre compte Discord si nécessaire
  3. Sélectionnez le serveur où vous voulez ajouter YAKO
  4. Accordez les permissions nécessaires
  5. Utilisez /setup pour configurer le bot

Quelles permissions YAKO a-t-il besoin ?

YAKO nécessite les permissions suivantes pour fonctionner correctement :

  • Gérer les messages - Pour la modération automatique
  • Gérer les rôles - Pour l'attribution automatique de rôles
  • Exclure des membres - Pour les sanctions
  • Bannir des membres - Pour les sanctions sévères
  • Envoyer des messages - Pour les réponses et notifications
  • Gérer les webhooks - Pour les logs avancés

Comment configurer l'auto-modération ?

Pour configurer l'auto-modération :

  1. Utilisez la commande /setup moderation
  2. Définissez les sanctions avec /automod penalties
  3. Configurez les filtres avec /automod filters
  4. Testez avec /automod test

Utilisation & Fonctionnalités

Comment voir toutes les commandes disponibles ?

Utilisez la commande /help pour voir toutes les commandes disponibles. Vous pouvez aussi utiliser /help [nom_commande] pour obtenir des détails sur une commande spécifique.

Comment créer des rôles automatiques ?

Pour configurer les rôles automatiques :

  1. Utilisez /autorole setup
  2. Sélectionnez le rôle à attribuer automatiquement
  3. Choisissez les conditions (nouveau membre, réaction, etc.)
  4. Confirmez la configuration

YAKO peut-il jouer de la musique ?

Oui ! YAKO dispose d'un lecteur musique intégré. Utilisez :

  • /play [url/titre] - Jouer une musique
  • /pause - Mettre en pause
  • /skip - Passer à la suivante
  • /queue - Voir la file d'attente
  • /stop - Arrêter la musique

Problèmes & Dépannage

YAKO ne répond pas à mes commandes

Si YAKO ne répond pas, vérifiez :

  • Que YAKO est en ligne (statut vert)
  • Que vous utilisez les slash commands (/commande)
  • Que YAKO a les permissions dans le salon
  • Que votre rôle est suffisamment élevé

Si le problème persiste, contactez le support.

Comment signaler un bug ?

Pour signaler un bug :

  1. Rejoignez notre serveur Discord de support
  2. Utilisez le salon #bug-reports
  3. Décrivez le problème en détail
  4. Incluez des captures d'écran si possible
  5. Mentionnez les étapes pour reproduire le bug

YAKO est-il gratuit ?

Oui, YAKO est entièrement gratuit ! Toutes les fonctionnalités de base sont disponibles sans frais. Nous offrons aussi un support premium pour les serveurs avec des besoins spécifiques.
